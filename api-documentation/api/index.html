Decimal number to roman number converter

This server takes in a number and outputs a roman numeral. Extension 1: Expand the set of numbers to 1-3999. Extension 2: Expand the set of numbers to include the Vinculum addition using UTF-8 to roman numerals greater than 3999 and handle all numbers up to 2,200,000,000.
