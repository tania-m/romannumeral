get

Numeral to roman converter route. Takes an integer (between 0 and 2200000000) as query parameter and converts it to a roman number. Example route: http://localhost:8080/romannumeral?query={integer} Success: returns status code 200 on success and the converted value; Error: returns status code 422 with error code and error message.
