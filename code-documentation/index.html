ConversionUpperLimits

Enum for conversion upper limits. SMALL: { name: "0->255", limit: 255 }: upper limit is 255 (included), MEDIUM: { name: "0->3999", limit: 3999 }: upper limit is 3999 (included), LARGE: { name: "0->2200000000", limit: 2200000000 }: upper limit is 2200000000 (included).

ConversionUpperLimits

Type: {name: string, limit: integer}

ConversionErrorCodes

Enum for error codes. NOT_AN_INTEGER: 'NOT_AN_INTEGER': the value is not an integer/cannot be converter to an integer, OUT_OF_RANGE: 'OUT_OF_RANGE': the value is not in the supported conversion range, VALUE_IS_ZERO: 'VALUE_IS_ZERO': the value to convert is 0 (no 0 in roman litterals).
